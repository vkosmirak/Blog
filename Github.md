###Revert specific commit

`git revert <commit>`


###Revert last 3 commits 

`git reset --soft HEAD~3`


###Apply current changes in another branch

```
git stash
git checkout other-branch
git stash pop
```

###Copy commit from another branch

```
git cherry-pick <commit>
```
