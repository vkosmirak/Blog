# Tips & Tricks

### Emails for testing
You can use `username@gmail.com` multiple times for registration.
Here is how:
```
username+1@gmail.com
username+2@gmail.com
username+lvl1@gmail.com
```
All letters will be redirected to `username@gmail.com`
